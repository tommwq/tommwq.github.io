JavaDoc使用方法

从源代码生成JavaDoc文档

javadoc -d 文档输出目录 源代码文件

JavaDoc提取规则

使用以下格式声明的注释，会被JavaDoc提取生成文档。 

/** ... */

/**
 * ...
 */

在生成文档时，注释中前导字符"*"、" "和"*"会被忽略。

JavaDoc标记

JavaDoc支持的标记有

标记 语法 说明 示例
@author @author 作者名 标记作者 @author fool@awesome.com
@version @version 版本号 标记版本号 @version 1.0
@see @see 类名 引用参考 @see java.awt.Button
  @see 类名#方法名   @see Fool#display(boolean)
  @see #方法名   @see main(String[])
@param @param 参数名 描述 描述方法的入参 @param username 用户名
@return @return 描述 描述方法返回值 @return result
@throws @throws 异常 描述 描述方法可能抛出的异常 @throws IllegalArgumentException 参数非法

规则

每个类、构造函数、公开函数必须以JavaDoc描述其用途和用法。
