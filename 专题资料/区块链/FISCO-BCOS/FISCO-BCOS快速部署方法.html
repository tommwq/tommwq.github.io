FISCO BCOS部署

检查前置条件

  • CPU 1.5GHz及以上
  • 内存 1GB及以上
  • 核心 2核及以上
  • 带宽 1Mb及以上
  • 操作系统 CentOS 7.2 64位 或 Ubuntu 16.04 64位
  • 软件
    • git
    • yum或apt
    • nodejs 6.0及以上
    • g++
cat /proc/cpuinfo | grep "cpu MHz"
cat /proc/cpuinfo | grep "cpu cores"
cat /etc/issue
cat /etc/lsb-release

下载FISCO BCOS软件包

git clone https://github.com/fisco-bcos/fisco-bcos

构建

cd fisco-bcos
chmod +x build.sh
./build.sh

如果输出

fisco-bcos build succ! path: /usr/local/bin/fisco-bcos

说明构建成功。

运行

cd sample
chmod +x run.sh
./run.sh

验证节点

ps -ef | grep fisco-bcos

验证节点连接

cat /bcos-data/node0/log/* | grep "Connected to 1 peers"

如果有输出，说明连接成功。

验证共识

tail -f /bcos-data/node0/log/* | grep ++++

如果有输出，说明共识成功。

部署合约

cd tool
babel-node deploy.js HelloWorld

调用合约

cd tool
bable-node demoHelloWorld.js

说明

  • 执行build.sh时，下载的软件包较多，需要下载大量NodeJS包，需要下载并构建包括Boost库在内的大量cpp包。需要的时间较长。
  • 在Ubuntu 17.10上没有nodejs-legacy包。需要改为nodejs。
