JavaScript简介

一个完整的JavaScript实现应该包括

  • 核心（ECMAScript）
  • 文档对象模型（DOM）
  • 浏览器对象模型（BOM）

在HTML中使用JavaScript

向HTML页面插入JavaScript的主要方法是使用<script>元素，它有6个属性：

  • async。可选。表示立即下载脚本，但不应阻塞页面中的其他操作。只对外部脚本文件有效。
  • charset。可选。表示外部脚本文件使用的字符集。
  • defer。可选。表示脚本延迟到文档被完全解析和显示后执行。只对外部脚本文件有效。
  • language。已废弃。
  • src。可选。指定外部脚本文件名。
  • type。可选。建议使用application/x-javascript或text/javascript。

<script>元素内的JavaScript代码被顺序解释。在值被计算完毕之前，页面中的其他内容不会显示。

传统做法将<script>元素放到<head>元素中，这样在显示页面时，所有脚本都已经加载完成。现代做法是将<script>放到<body>元素的尾部，这样可以缩短首次渲染时间。

如果声明了延迟（defer="defer"），<script>元素会在浏览器遇到</html>后执行。HTML5要求延迟<script>按照元素的排列顺序执行，并且在DOMContentLoaded事件之前执行。但浏览器的实现未必满足HTML5要求，因此延迟<script>最好不要超过1个。

异步脚本（async="async"）的加载和执行顺序由浏览器决定，异步脚本在页面的load事件前执行，但不保证在DOMContentLoaded事件前执行。异步脚本不要在加载器修改DOM。

很多浏览器提供了混杂模式（quirks mode）、准标准模式（almost standards mode）和标准模式（standards mode），这些模式依次更加符合标准。

基本概念

严格模式

ECMAScript 5引入了严格模式（strict mode）的概念。严格模式对早前版本中的危险行为做了处理。要使用严格模式，可以在脚本顶部添加 

"use strict";

如果只希望在某个函数内部使用严格模式，可以在函数体内加入编译指示： 

function doSomething() {
    "use strict";
    // ...
}

数据类型

ECMAScript提供了5种基本数据类型和1种复杂类型，它们是

  • Undefined
  • Null
  • Boolean
  • Number
  • String
  • Object

typeof操作符可以返回一个对象的类型，它的返回值是

Undefined "undefined"
Boolean "boolean"
String "string"
Number "number"
Object "object"
函数 "function"
Null "object"
未声明变量 "undefined"

Undefined类型只有一个值：undefined。所有已声明未赋值的对象都具有这个值 

var message; // 等效于 var message = undefined;

要注意，未声明的对象不具有undefined值。

Null类型也只有一个值：null。null表示一个空引用。undefined继承自null： 

alert(null == undefined); // true

Boolean类型包含2个值，true和false，分别表示真和假。Boolean可以从其他对象转换得来，如： 

var message = "hello";
var messageAsBoolean = Boolean(message);

转换的规则如下

类型 转换为true的值 转换为false的值
Boolean true false
String 非空字符串 空字符串
Number 非零数值（包括无穷大） 0和NaN
Object 任何对象 null
Undefined 不适用 undefined

Number类型是IEEE754双精度浮点数。Number可以表示的最大和最小数是Number.MAXVALUE和Number.MINVALUE。如果计算结果超出这个范围，值会被设置为Infinity（正无穷大）或-Infinity（负无穷大）。函数isFinity()可以判断一个数是否超过有效范围。Number中有一个特殊的值NaN。任何数除0都会产生NaN，对NaN进行任何运算结果还是NaN，还有，NaN和任何值（包括NaN自身）进行比较都不相等。 

alert(NaN == NaN); // false

函数isNaN()可以判断一个对象是不是（或能不能转换为）数值。

有3个函数可以将对象转换为Number类型：Number()、parseInt()和parseFloat()。Number()的转换规则为

类型 对象 数值
Boolean true 1
  false 0
Null null 0
Undefined undefined NaN
String 0x前缀字符串 对应的10进制值
  浮点数字符串 对应值
  整数字符串 对应值，忽略前导的0
  空字符串 0
  其他字符串 NaN
Object   将valueOf()值转换为Number。如果结果为NaN，使用toString()值进行转换
