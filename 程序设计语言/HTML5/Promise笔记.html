#+title Promise笔记 #+date 2017年01月06日

Promise是一个容器，封装了一个异步操作。

语法

new Promise(executor);
new Promise(function(resolve, reject) {...});

这里executor是用于参数resolve和reject的函数。在创建Promise对象的同时，executor已经开始执行。 executor执行成功后会调用resolve函数。如果executor执行失败，reject函数会被调用。

状态

Promise对象可能处于以下3中状态之一

  • pending 初始状态
  • fulfilled 执行成功
  • rejected 执行失败

在executor执行过程中，Promise对象处于pending状态。executor执行完毕后，对象会根据执行结果转换为fulfilled或rejected状态。

状态准换

Promise对象提供了方法then()处理状态转换事件。 当Promise对象的状态发生变化时，通过then()绑定的方法立即被调用。 如果在绑定方法时，Promise对象已经处于fulfilled或rejected状态，被绑定的方法立即执行。

属性

  • Promise.length 构造器参数数量，值为1。
  • Promise.prototype 构造器原型。

方法

  • Promise.all(iterable)
  • Promise.race(iterable)
  • Promise.reject(reason)
  • Promise.resolve(value)

Promise原型

属性

  • Promise.prototype.constructor

方法

  • Promise.prototype.cache(onRejected) 在原有Promise对象上绑定reject回调函数，作为新Promise对象返回。
  • Promise.prototype.then(onFullfilled, onRejected) 在原有Promise对象上绑定回调函数，作为新Promise对象返回。

示例

<div id=log></div>
<button onclick="testPromise();">TestPromise</button>
<script>
'use strict';
var promiseCount = 0;
function testPromise() {
    var thisPromiseCount = ++promiseCount;
    var log = document.getElementById("log");
    log.insertAdjacentHTML("beforeend", thisPromiseCount + ") 开始（同步代码开始）<br/>");
    var p1 = new Promise(function (resolve, reject) {
        log.insertAdjacentHTML("beforeend", thisPromiseCount + ") Promise开始（异步代码开始）<br/>");
        window.setTimeout(function () {
            resolve(thisPromiseCount);
        }, Math.random() * 2000 + 1000);
    });

    p1.then(function (val) {
        log.insertAdjacentHTML("beforeend", val + ") Promise被满足了（异步代码结束）<br/>");
    });

    log.insertAdjacentHTML("beforeend", thisPromiseCount + ") 建立了Promise（同步代码结束）<br/><br/>");
}
</script>
