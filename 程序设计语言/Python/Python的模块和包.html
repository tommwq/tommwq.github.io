Python中的模块和包

模块

在Python中，一个.py文件就是一个模块。

包

包由多个模块组成。这些模块放在同一个目录下，并且在目录下需要有一个_init_.py文件。包中导出的名字需要声明在变量_all_中。下面是一个示例： 

# __init__.py
__all__ = ["module1", "sub_package1", "function1"]

导入顺序

python 包的导入顺序是

  1. 系统包。
  2. 同目录。
  3. sys.path。

导入建议

import os
import sys

def _dirname(path, up_level=1):
    result = os.path.abspath(path)
    for i in range(0, up_level):
        result = os.path.dirname(result)
    return result

sys.path.append(_dirname(__file__, n)) # n是目录深度
