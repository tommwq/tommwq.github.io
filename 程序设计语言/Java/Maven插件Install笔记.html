Maven插件Intall

Install插件用于在构建活动的install阶段，将软件包安装到本地仓库。本地仓库位于目录~/.m2/repository，或配置文件~/.m2/settings.xml中<localRepository>标签指向的目录。

目标

Install插件包含3个目标：

  • install:install 将Maven工程的主要构件（artifact）安装到本地仓库。
  • install:install-file 将文件安装到本地目录。
  • install:help 查看帮助信息。

用法

  1. 安装Maven工程到本地仓库
mvn install
  1. 安装文件到本地仓库
mvn install:install-file -Dfile=name_in_repo.jar \
        -DpomFile=pom.xml \
        -Dsources=path_to_file.jar \
        -Dpackagine=jar \
        -DgroupId=some.group \
        -DartifactId=artifact \
        -DVersion=1.0 \
        -DgeneratePom=true \
        -DcreateChecksum=true

Install插件和Deploy插件的区别

Install插件用于将构件保存到本地仓库。如果需要将构件上传到远程仓库，需要使用Deploy插件。
