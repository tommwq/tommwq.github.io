获得帮助

对于不熟悉的命令，可以用doc命令查看文档： 

doc randi

运算结果

如果没有指定保存计算结果的变量，matlab会将结果保存在变量ans中。在命令尾部增加分号“;”可以抑制输出。命令clear可以清除所有变量，命令clc可以清除命令行窗口。

矩阵和向量

matlab的向量用方括号“[]”包围。如果元素之间以空格“ ”分隔，这是一个行向量。如果元素以分号“;”分隔，这是一个列向量。 

rvec = [1 2 3]
cvec = [1;2;3]

类似的，方括号也可以用来表示矩阵。下面是一个2行3列的矩阵： 

m = [1 2 3; 4 5 6]

matlab为创建等距行向量提供了便捷的语法，如： 

x = 1:3 % x=[1 2 3]
x = 1:2:5 % x=[1 3 5]
x = linspace(0,4,2) % x=[0 4]

上面这些都是创建行向量的。如果需要列向量，可以通过转置完成： 

x = x'
x = (5:2:9)'

对于一些特殊的常用矩阵，Matlab也提供了方法： 

rand(n[,m=n]) %创建一个n*m的随机数矩阵。
zeros(n[,m=n]) %创建一个n*m的0矩阵。

矩阵的元素可以通过下表获得 

v=m(2,3)

如果选择最后一行/列，可以使用关键字end。end支持运算： 

v=m(end,3)
u=m(end-1,3)

如果使用“:”作为下表，可以对矩阵进行切片： 

x=A(2,:) % x是A的第二行
y=A(:,2) % y是A的第二列
z=A(1:3,:) % z是A的1-3行组成的子矩阵

对于向量，使用单个索引可以引用其中的元素： 

p = vec(3)

使用下表可以为元素赋值： 

vec(2) = 1

当标量和向量相加时，向量的全部元素都会加上这个标量。两个大小相等的向量相加，是其元素相加。对于矩阵，“*”是矩阵乘法。按元素乘法是“.*”。

要得到矩阵的行数和列数，可以使用size()函数。它有多个返回值，第一个是矩阵的行数，第二个是列数： 

[dr,dc] = size(data)

matlab允许函数返回多个数值。比如函数max()的第一个返回值是最大值，第二个返回值是最大元素的索引。对于多返回值函数，用“[]”可以将这些返回值依次赋值给多个变量。如果只取部分返回值，其余的值被丢弃。

numel(A)数组元素数目。等于prod(size(A))。

保存和加载数据

命令 

save foo x

将数据x保存到foo.mat文件中。而 

load foo

会从foo.mat中加载数据。

绘制图形

plot命令可以绘制图形： 

plot(x, y)
plot(x, y, 'r--o') % 绘制红色虚线，并用“o”标记。-.点划线--虚线:点线
plot(x,y,'r*') % 绘制红色“*”标记，不画线。
plot(x,y,'ks') % 使用方块（k）标记。
plot(v1,'LineWidth',5) % 线宽。

每次调用plot时会清除上次绘制的图形。如果要保留图形，可以输入 

hold on

如果要关闭绘图窗口，可以执行命令 

close all

title("ssss")可以为绘图设置标题。ylabel("dd")可以设置y轴标题。xlabel是x轴标题。legend('a','b','c')可以增加图例，表明不同的线段对应哪个变量。

fft(x)离散傅里叶变幻。

matlab的和于或运算符是&和|。

下表也可以用于过滤： 

x=v(v>6)  % x是v中大于6的元素组成的子矩阵。
y=A(B<6) % 要求A和B的维度相同。
v1(v1>5) = 10 % 将v1中大于5的值替换为10。

分支： 

x = rand
if x > 0.5
  y = 3
end

if x > 0.5
  y = 3
else
  y = 4
end

循环： 

for i = 1:3
  disp(i)
end
