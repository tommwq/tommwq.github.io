神经网络与深度学习笔记

深度学习（deep learning）是机器学习的一个分支。 机器学习尝试从有限的样例中总结出一般性规律，并应用到未知数据上。 深度学习中，贡献度分配（Credit Assignment Problem，CAP）是一个关键问题。 人工神经网络（Artificial Neural Network，ANN）对CAP问题解决的较好，因此深度学习往往采用ANN实现。

人工智能主要分为感知、学习、认知三个方面。感知主要是对样本进行分类，包括语音信息处理和计算机视觉。 学习是一种优化模型的手段。认知是理解信息之间的逻辑关系。

专家系统（Expert System）主要由知识库和推理机构成。

ANN可以看成是一个函数逼近器。一个两层的ANN可以逼近任意函数。 一个ANN可以塑造的函数叫做网络容量（Network Capacity）。

BP：Backpropagation。

机器学习的目标是得到一个预测模型。机器学习的数据首先要抽象为一组特征（feature）。机器学习的步骤如下：

  • 数据预处理。去除噪声。
  • 特征提取。
  • 特征转换。常用的方法有主成分分析（Principal components analysis，PCA）和线性判别分析（Linear Discriminant Analysis）等。
  • 预测。

有效特征也叫做表示（representation）。如果一个算法可以自动的学习有效特征，这种方法久叫做表示学习（representation learning）。 表示学习的关键是解决语义鸿沟（semantic gap）。语义鸿沟是输入数据的底层特征和高层语义信息之间的差异。 在表示学习中有两个核心问题：什么是一个好的表示？如何学习到好表示？

“深度”是对原始数据进行进行转换的次数。表示学习需要深度学习。 深度学习的目标是从数据中自动学习得到有效的特征表示。

深度学习采用误差反向传播算法进行参数学习。

按照学习的方法，机器学习可以分为监督学习、无监督学习、强化学习。

神经网络分为：前馈神经网络、卷积神经网络、循环神经网络、图网络。

概率图模型：玻尔兹曼机、深度信念网络。 概率生成模型：深度生成模型、序列生成模型。

特征工程：设计不同的特征及特征组合。

特征也叫属性（attribute）。样本（sample）也叫示例（instance）。

样本包括一组特征和一组标签（label）。标签是对样本的评价。样本的集合叫做数据集（data set）。 在进行机器学习时，通常将数据集分为两个部分，一部分用于训练模型，叫做训练集（training set）和训练样本（training sample），另一部分 用于检验模型效果，叫做测试集（test set）和测试样本（test sample）。

假设样本拥有一个d维的特征向量（feature vector）x=[x1, x2, …, xd]T ，训练集有N个样本，每个样本是独立同分布（Identically and Independently Distributed，IID）的，记为 D = { (x(1), y(1)), …, (x(N), y(N)) } }

机器学习的目标就是寻找一个决策函数f(x;θ)。有了这个决策函数，就可以预测标签的值

y = f(x;θ)

或得到标签的条件概率

p(y|x) = fy(x; θ)

这里的 θ 就是可学习参数。在训练集上找到一组函数 θ*，使得函数f(x, θ*)贴近真实的映射关系， 这个过程叫做学习（learning）或训练（training）。函数f(x;θ)也叫做模型（model）。

习得一个模型（参数）后，需要在测试集中检验模型的效果。

机器学习方法拥有3个基本要素：模型、学习准则和优化算法。

进行机器学习之前首先要明确输入空间X和输出空间Y（标签空间）。不同机器学习任务的主要区别是输出空间不同。 对C类分类问题，输出空间为ZC，对回归问题，输出空间为R。

输入空间X和输出空间Y构成了一个样本空间。假设存在一个函数g: X->Y，使得y=g(x)， 或者存在一个条件概率分布Pr(y|X)。机器学习的目标就是逼近这个函数（或条件概率分布）。 至于函数（或条件概览分布）的存在性，不是机器学习关注的范围。

学习的过程就是就是基于训练样本构造一组近似函数。这组近似函数的集合叫做假设空间（hypothesis space），记为F。 然后从中选择一个最优的假设（hypothesis）。通常假设空间是一个参数化函数族 F = {f(x;θ) | θ ∈ Rm} 其中f(x;θ)也叫做模型， θ 是可学习参数，m是参数的数量（或维度）。

假设空间可以是线性的或非线性的。对应的模型分别叫做线性模型和非线性模型。

线性模型可以表示为f(x;θ) = wT x + b，参数 θ 包含了权重向量w和偏置b。

非线性模型可以表示为多个非线性模型的线性组合： f(x;θ) = wT Φ(x) + b 其中Φ(x) = [φ1(x), …, φK(x)]T是K个非线性基函数组成的向量。参数 θ 包含了权重w和偏置b。

如果Φ(x)中的基函数也是可学习的，比如 φk(x) = h(wkT Φ^'(x) + bk) 其中h()是非线性函数，φ^'(x)是另外一组基函数，wk和bk是可学习函数，那么f(x;θ)就是神经网络。

学习准则。假设训练集 D={(x(n),y(n)}n=1N 是N个独立同分布（IID）的。这里要用Pr(x,y)不能是时变的，否则无法进行学习。

一个好的模型应当在所有样本(x,y)上都足够贴近真实函数，即 abs(f(x,θ*) - y) < ε 或 abs(fy(x,θ*) - pr(y|x)) < ε

模型的好坏可以通过期望错误（expected error） R(θ) 来衡量 R(θ) = E(x,y) ~ pr(x,y)[L(y,f(x;θ))]

损失函数（loss function）是一个非负实值函数，用来描述预测和实际之间的差异。常见的损失函数有： 0-1损失函数。 \[ L(y,f(x;\theta) = I(y \neq f(x;\theta)) \]

平方损失函数（quadratic loss function）。通常用于标签为实数的情况，不适用于分类问题。 \[ L(y,f(x;\theta)) = \frac{1}{2} (y - f(x;\theta))^2 \]

交叉熵损失函数（cross-entropy loss function）。一般用于分类问题。假设样本标签 \[ y \in {1, ..., C} \] ，模型 \[ f(x;\theta) \in [0,1]^C \] 输出一个条件概率分布 \[ p(y=c|x; \theta) = f_c(x; \theta) \] 并满足 \[ f_c(x;\theta) \in [0,1] \sum_{c=1}^C f_c(x;\theta) = 1 \]

Hinge损失函数。Hinge损失函数适用于两类分类问题。

最小错误原则。一个好的模型应当具有比较小的期望错误。由于不了解实际的数据分布 和映射函数，只能根据训练集中的样本进行计算。 这个结果叫做经验风险（Empirical Error）， 这个原则叫做经验风险最小化（Empirical Risk Minimization）原则。

过拟合overfitting。对于训练集D，给定假设空间F中的两个假设f和f'， 如果f在D上的损失比f'小，而在整个样本空间上f'的损失比f小，那么就说明假设f 过度拟合训练集。

过度拟合的原因：训练集小、噪声、模型能力强。对于模型能力强的问题，可以进入参数 正则化以限制模型能力，及结构风险最小化（structure risk minimiation，SRM）原则。

欠拟合underfitting。模型能力不足。

优化算法。一旦训练集D、假设空间F和学习准则确定后，找到最优模型就变成了一个 最优化问题。在训练模型的过程中，我们可以调整优化策略参数，这类参数叫做超参数（hyper-parameter）， 常见的有聚类个数、梯度下降步长、神经网络层数等。

梯度下降算法。迭代计算风险函数最小值。 \[ \theta_{t+1} = \theta_t - \alpha \frac{\partial R_D(\theta)}{\partial \theta} \]
