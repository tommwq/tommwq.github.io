检查运行环境

CPU

必须使用 x86_64 架构处理器。 

uname -m

Linux内核版本

Linux内核版本不低于3.10。 

uname -r

iptables

iptables版本不低于1.4。 

iptables --version

git

git版本不低于1.7。 

git --version

ps

提供ps或procps命令。

XZ Utils

XZ Utils版本不低于4.9。 

xz --version

cgroupfs

cat /proc/cgroups

下载Docker二进制文件

https://github.com/docker/docker/releases

验证下载文件

https://get.docker.com/builds/Linux/x86_64/docker-1.11.0.tgz.md5 https://get.docker.com/builds/Linux/x86_64/docker-1.11.0.tgz.sha256

安装Docker

tar -xvzf docker-latest.tgz
mv docker/* /usr/bin/

启动Docker进程

sudo dockerd &

设置开机启动

sudo systemctl enable docker

或在/etc/rc.local中加入 

nohup dockerd 1>/dev/null 2>&1 &

设置组和用户

建立docker组

groupadd docker

将用户加入docker组

usermod -aG docker <username>

然后重新登录，使用户组配置生效。
