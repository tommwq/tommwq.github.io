jfr和jmc简介

jfr（Java Flight Recorder）和jmc（Java Mission Control）是JDK提供的两个工具，jfr用于记录JVM的运行情况，jmc将记录的数据展示出来。结合jfr和jmc，可以方便的定位Java程序中的性能瓶颈。

使用jfr记录JVM运行情况

jfr需要JDK商业证书，在使用前，必须进行解锁： 

jcmd <PID> VM.unlock_commercial_features

之后就可以使用jfr记录JVM信息： 

jcmd <PID> JFR.start name=test duration=60s settings=template.jfc filename=output.jfr

下面的方法可以在启动JVM时开启jfr： 

-XX: +UnlockCommercialFeatures -XX: +FlightRecorder -XX:+FlightRecorderOption=参数
