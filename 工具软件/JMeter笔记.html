使用JMeter测试Java服务吞吐量

建立测试项目

引入JMeter

<dependency>
     <groupId>org.apache.jmeter</groupId>
     <artifactId>ApacheJMeter_java</artifactId>
     <version>5.1</version>
</dependency>

编写测试程序

package com.guosen.test.jmeter.test;

import org.apache.jmeter.config.Arguments;
import org.apache.jmeter.protocol.java.sampler.AbstractJavaSamplerClient;
import org.apache.jmeter.protocol.java.sampler.JavaSamplerContext;
import org.apache.jmeter.samplers.SampleResult;

public class TestHello extends AbstractJavaSamplerClient {
        HelloRequest req = new HelloRequest();

        @Override
        public void setupTest(JavaSamplerContext arg0) {
        }

        public Arguments getDefaultParameters() {
                Arguments params = new Arguments();
                return params;
        }

        public void setValues(JavaSamplerContext arg0) {
        }

        @Override
        public void teardownTest(JavaSamplerContext arg0) {
        }

        @Override
        public SampleResult runTest(JavaSamplerContext arg0) {
                SampleResult sr = new SampleResult();
                sr.sampleStart();  
                try{
                        callService();
                        sr.setSuccessful(true); 
                }catch(Exception e){
                        sr.setSuccessful(false);
                        e.printStackTrace();
                }finally {
                        sr.sampleEnd();
                }
                return sr;
        }
}

打包

maven打包生成的普通jar包，只包含该工程下源码编译结果，不包含依赖内容。同时，maven提供以下方式生成包含所有依赖的jar文件，依赖以class的方式存在； 将此plugin添加到pom文件中 在pom.xml中加入插件 

 <plugin>
    <artifactId>maven-assembly-plugin</artifactId>
    <configuration>
        <!--这部分可有可无,加上的话则直接生成可运行jar包-->
        <!--<archive>-->
            <!--<manifest>-->
                <!--<mainClass>${exec.mainClass}</mainClass>-->
            <!--</manifest>-->
        <!--</archive>-->
        <descriptorRefs>
            <descriptorRef>jar-with-dependencies</descriptorRef>
        </descriptorRefs>
    </configuration>
</plugin>

然后执行 

mvn assembly:assembly

测试

复制jar包

将-jar-with-dependencies结尾的JAR包放到%JMETERHOME%\lib\ext下。

启动JMeter

jmeter.bat -n -t NAME
