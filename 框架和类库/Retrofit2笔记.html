Retrofit2笔记

简介

Retrofit可以简化HTTP API调用。通过Retrofit，调用HTTP API变得想调用本地方法一样简单。

安装

implementation 'com.squareup.retrofit2:retrofit:2.5.0'

用法

我们已github.com提供的users/{user}/repos接口为例子。首先，根据这个HTTP API，建立一个接口： 

public interface GitHubService {
    @Get("users/{user}/repos"
    Call<List<Repo>> listRepos(@Path("user") String user);
}

然后建立Retrofit对象，并通过Retrofit建立HTTP API的代理。 

Retrofit retrofit = new Retrofit.Builder()
    .baseUrl("https://api.github.com")
    .addConverterFactory(GsonConverterFactory.create()) // retrofit2.converter.simplexml.SimpleXmlConverterFactory
    .build();
GitHubService service = retrofit.create(GitHubService.class);

接下来就可以想本地方法一样调用了 

Call<List<Repo>> repos = service.listRepos("octocat");

Call支持同步和异步两种方式： 

Response<T> call.execute(); // 同步
void call.enqueue(Callback<T> cb); // 异步

在异步情况下，Callback提供两个方法： 

void onFailure(Call<T> call, Throwable t);
void onResponse<Call<T> call, Response<T> resp);

Response是接口返回的数据，常用方法有： 

T body();
String message();

Retrofit自身比提供对json和xml的解析，不过可以通过converter来完成。 

Retrofit retrofit = new Retrofit.Builder()
    .baseUrl("http://api.github.com")
    .addConverterFactory(GsonConverterFactory.create())
    .build();

常用的converter有：

Gson: com.squareup.retrofit2:converter-gson:2.5.0 Jackson: com.squareup.retrofit2:converter-jackson:2.5.0 Protobuf: com.squareup.retrofit2:converter-protobuf:2.5.0 Simple XML: com.squareup.retrofit2:converter-simplexml:2.5.0 JAXB Converter: com.squareup.retrofit2:converter-jaxb:2.5.0

jaxb不支持Android。

在Android上使用Simple XML Converter时，需要排除stax:stax-api, stax:stax, and xpp3:xpp3。 

implementation('com.squareup.retrofit2:converter-simplexml:2.5.0') {
    exclude group: 'stax', module: 'stax-api'
    exclude group: 'stax', module: 'stax'
    exclude group: 'xpp3', module: 'xpp3'
}

Simple XML Converter使用了org.simpleframework.xml。

http://simple.sourceforge.net/download/stream/doc/tutorial/tutorial.php
